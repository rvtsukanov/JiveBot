Проекты, по которым департаментом маркетинга принято решение использовать в маркетинговых активностях, обязательно проходят согласование юристами и ДК.  Юристы, принимают решение на основании заключенных договоров и соглашений.  ДК принимает решение на основании взаимодействия с клиентом.  От согласующего ДК ожидается принятие решения о возможности использования проекта в маркетинговых активностях. ДК может принять одно из следующих решений:  1. проект не может участвовать в маркетинговых активностях, т.е. нельзя упомянуть участие заказчика в данном проекте, даже нельзя использовать содержание работ без наименования заказчика и т.д. Ничего нельзя. Такие проекты единичны. 2. проект может участвовать в маркетинговых активностях, т.е. мы можем использовать любые данные, которые есть по проекту в любой комбинации. 3. проект может участвовать с маркетинговых активностях с ограничениями, т.е. ДК принимает решение и подробно описывает в комментариях, какие данные нельзя использовать в пиар деятельности. (см. пример на рис. 1.)

Полезная информация по  работе с плагинами для MS Office : Плагины для Office и Outlook 2016 Плагины Jive для MS Office и Outlook Работа с документами в Jive Jive и электронная почта   Совместная работа над документом MS Office с применением Jive   Как устроенна групповая работа над документами     1) Как активировать свой плагин Сразу после установки плагина, необходимо его активировать. Чаще всего это делается автоматически, при совершении какой-либо операции с контентом. Пока это не произошло, ваш плагин горит красным   Для его активации нужно кликнуть на Учетные записи (Accounts)   И кликнуть на учетную запись  https://jive.croc.ru  для активации. Система предложит ввести логин и пароль. После этого ваши плагины будут активны и больше не будут просить логин и пароль.   2) Как правильно открывать файлы через плагины MS Office Правильно открывать файлы не через браузер, а через само приложение, нажав на кнопку открыть (Open). Если вы открываете через файл через браузер, то стоит его сначала сохранить, а потом уже осуществлять работу с содержимым.   Во всплывающем окне Вам будет предложено выбрать группу/пространство и документ для открытия. По умолчанию отображаются пространства и группы на которые вы подписаны, однако с помощью поиска можно найти любые доступные места и документы.   2) Как правильно сохранять/обновлять файлы через плагины MS Office   Сохранять или обновлять файлы можно с помощью соответствующих кнопок панели Jive Опубликовать новый документ  (Publish New) - при нажатии Вам будет предложено выбрать место для публикации документа Опубликовать как другой документ (Publish AS) - при нажатии Вам будет предложено выбрать документ из Jive, который вы хотите заменить. Ваш документ опубликуется как новая версия уже существующего документа   В случае если у вас уже открыт документ jive, Вас будет доступна кнопка Опубликовать обновлении (Publish Update), которая сохраняет изменения в открытом файле и обновляет его версию     Для браузеров Плагин для браузера - Google Chrome, Internet Explorer, Firefox, Chrome, Safari   Приложение: Мобильный клиент для Jive

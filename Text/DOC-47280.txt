 1 Проекты для пиара Не все проекты будут проходить согласование маркетингом. Но, если маркетинг решает пиарить проект, то данную возможность он должен согласовать с юристами и ДК. 1. существуют проекты, которые автоматически будут приходить на этап согласование маркетингом. К ним относятся  проекты, выполненные за последние 5 лет ; 2. остальные проекты, не будут автоматически переходить на этап согласования маркетингом. Такие проекты маркетинг самостоятельно может найти и начать согласование для участия в пиаре. Если по проектам в п.2 был стартован процесс сбора проектного опыта, то в разделе  Проектный опыт КРОК  в карточке проекта будет отображен статус (см. рисунок 1). Рисунок 1. - Карточка проектного опыта в Jive, требующая согласования для участия в маркетинговых активностях Следовательно, без согласования юристами и ДК проект не может быть использован в маркетинговых активностях.  Для редактирования данных по проекту и отправки на согласование для участия в маркетинговых активностях необходимо нажать на кнопку зеленого цвета "Изменить проектный опыт" (см. рисунок 1). Далее сотруднику маркетинга приходит нотификация на почту. 2 Согласование маркетингом Сотрудник маркетинга, которому пришла нотификация о сборе проектного опыта, может направить заполнение проекта другому сотруднику маркетинга, либо заполнить форму самостоятельно. Ответственный сотрудник может заполнить/изменить все поля на форме. Может изменить поле "Директор клиента" и выбрать сотрудника из справочника.  Решения, которые может принять сотрудник маркетинга (см. рисунок 2).   Рисунок 2. - Решения принимаемые сотрудником маркетинга 1. завершить процесс - сотрудник выбирает данное решения в случае, если нет необходимости, чтобы проект принимал участие в маркетинговых активностях. Если в дальнейшем будет принято решение пиарить данный проект, необходимо начать процесс изменения проектного опыта (см. рисунок 1); 2. вернуть менеджеру на корректировку - в случае если на форме представлено критически мало информации, сотрудник маркетинга может оставить свои комментарии на форме и выбрать данную резолюцию; 3. перенаправить сотруднику маркетинга - при выборе данного решения, появляется возможность выбора сотрудника из выпадающего списка. 4. отправить на согласование юристам для пиара - данное решение принимается, если информация на форме заполнена корректно и готова для использования в маркетинговых активностях. В данном случае оно обязательно. 3 Согласование юристами Юристы, изучив документацию по проекту, принимают решение о том, можно ли использовать проект в маркетинговых активностях. Юристы могут принять одно из трех возможных решений: 1. проект не может участвовать в маркетинговых активностях, т.е.   нельзя  упомянуть участие заказчика в данном проекте, нельзя использовать содержание работ без наименования заказчика и т.д.  Маркетингу про него надо забыть.  Такие проекты единичны. 2. проект может участвовать в маркетинговых активностях, т.е. мы можем использовать любые данные, которые есть по проекту в любой комбинации. 3. проект может участвовать с маркетинговых активностях с ограничениями, т.е. ответственный юрист принимает решение и подробно описывает в комментариях, какие части согласно договору нельзя использовать в пиар деятельности. Также, если в договоре есть срок окончания данного договора, то ответственный юрист также укажет в комментариях. Варианты решения аналогичны вариантам решений, принимаемых юристами.

При первой установке плагина для MS Office указывается путь по умолчанию, куда должны сохраняться файлы при их открытии с помощью плагина для MS Office. Может кому-то будет полезна информация, как этот путь изменить.     Путь сохранен в файле настроек  AppSettings.config   \Users\User name\AppData\Roaming\OffiSync    У меня этот файл выглядел так:   <?xml version="1.0" encoding="utf-8"?> <UserSettings>   <Settings>     <Setting name="DownloadAfterCoAuthoringPopup" value="True" />     <Setting name="EnableSavingOnlineFinished" value="True" />     <Setting name="InstallationId" value="d8b08f" />     <Setting name="DownloadToFolder" value= "D:\Jive" />     <Setting name="CurrentAssemblyVersion" value="20.1.395.13521" />     <Setting name="PostComponentCleanupVersion" value="1" />     <Setting name="OutlookSocialBarState" value="0" />     <Setting name="SuperContextSearchProviderDisabled" value="True" />   </Settings> </UserSettings>

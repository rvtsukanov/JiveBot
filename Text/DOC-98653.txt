Для удобства и структурирования контента в стандартном функционале форматирования текста  Документа  предусмотрена вставка таблицы. Этот стандартный функционал обладает набором возможностей.   Вставка таблицы В шапке стандартного набора форматирования текста документа нажмите на изображение "Вставить таблицу" Настройки таблицы Нажмите на изображение "Редактировать таблицу"  А так же можно редактировать Стобец, Строку, Ячейку, Слить строки/столбцы таблицы, Удалить таблицу: Нажав на "Редактировать таблицу", можете выбрать Стиль  таблицы: Изменить положение  заголовков Ширину ( в пикселях) и Цвет  границ Размер  по ширине и высоте в соотношении пикселей процентном автоматически положение без выравнивания по левому краю по правому краю Настройки  Заголовка   Образец заголовка Стиль:  отступ/фон Текст: Выравнивание заголовка (слева, по центру, справа, по ширине); Положение заголовка ( по верхнему краю, по центру, по нижнему краю) Семейство шрифтов (Arial, Calibri и др.) Цвет текста заголовка Настройки  Ячейки Образец ячейки Стиль ячейки Отступ (в пикселях) Фон ячейки (можно выбрать для выделения контента) Текст Выравнивание ячейки (слева, по центру, справа, по ширине); Положение ячейки ( по верхнему краю, по центру, по нижнему краю) Семейство шрифтов (Arial, Calibri и др.) Цвет текста ячейки Нажав на значок "Настройки" можно добавить/удалить строку/столбец таблицы. А так же, для быстрого выравнивания Таблицы по центру, перейдите в  "Включение/ выключение режима HTML"   в шапке стандартного набора форматирования текста документа затем пропишите строчки в коде выделенные  красным цветом   <body>  <center> .... </center> </body>

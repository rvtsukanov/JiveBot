   1. Создание документа Чтобы создать документ, можно в шапке страницы jive в правом верхнем углу найти инструмент "создать"  , выбрать Документ. 2.  Стандартные возможности форматирования текста Оформите документ, используя стандартный набор возможностей форматирования текста: Текст и его шрифт:  Выбор шрифта, Размер шрифта, Начертание к тексту (Полужирный, Курсив, Подчеркивание, Зачеркнуть текст), Цвет,  Стиль  (Параграф/  Заголовок 1,2 ... ), Убрать форматирование Оформление текста:  Маркированный список, Упорядоченный список, Уменьшить/Увеличить отступ Выравнивание:  по левому краю/ по центру/ по правому/ по ширине   3. Вставка в тексте Видео : Введите URL видео Код вставки Изображение: С компьютера (Выберите файл) Из интернета (Введите URL-адрес изображения) Вставить ссылку/Гиперссылку или Добавить материал Добавить гиперссылку к выделенному тексту/изображению:  добавление  Веб-адреса (по URL) Добавить материал через поиск в системе:   Все материалы Добавить материал из последних открытх материалов:   Просмотреть историю Таблица Более подробно:  Настройки таблицы для Документа в jive Проверка орфографии Смайлики          Вставка  Оглавление Оглавление помогает в навигации по тексту по  Заголовкам  разного уровня, соответственно структурировать текст Например:   Для создания оглавления требуется  выделить нужные заголовки, соответственно с их структурной значимостью уровня, т. е.  Заголовок 1 (1 уровень), Заголовок 2 (второй уровень) и т.д. Подсветка синтаксиса (XML, SQL, Java и др.) Вставить неформатированный HTML-код Цитата Напоминание (успех/Информация/предупреждение/опасность) Пометить (Новое/Обновлено)   4. Добавить: Упоминание, Прикрепить, Приложение @Упоминание  (Люди/Места/Материалы) Помогает быстро упомянуть в тексте нужное место или материал, а так же привлечь внимание коллеги к данному документу. После написания "@ ", начните вводить нужное название материала/места или имя пользователя. Прикрепить файл (с компьютера) Позволяет добавить вложение к документу, тем самым не перегружая документ другим контентом. ! Приложение 5. Дополнительные опции : модерация и ограничение доступа В нижней части страницы перед публикацией документа Вы можете использовать  Дополнительные опции   Созданный документ можно опубликовать:  В месте  (группе/разделе),  Скрыто  (скрыть ото всех, при этом он будет в ваших материалах), с доступом для  Конкретных люде й, в  собственных материалах В соответствии с местом публикации, можно настроить дополнительные опции документа: Ограничить авторов  Только я могу редактировать этот документ  (В месте; Конкретные люди) . Для ограничения редактирования в  "Моих материалах"   Просто  не нужно  выставлять Дополнительных опций. Разрешить определенным лицам редактировать этот документ, введя их инициалы в поиске и последующем добавлении ( В месте ; для  Конкретных людей ) Разрешить определенным лицам редактировать данный документ из  "Моих материалов" Модерация Утверждение   указанного пользователя   перед публикацие й  внесенных изменений   обязательно   (В месте) Ограничить комментарии  закрыто для всех (комментарии не будут отображаться; добавление новых комментариев запрещено)  ( В месте ; для  Конкретных людей ;  Мои материалы ) закрыто (старые комментарии отображаются, новые комментарии запрещены)  ( В месте ; для  Конкретных людей ;  Мои материалы )   Привлекайте внимание конкретной целевой аудитории или организуйте свои записи по теме или по группе. Не показывать материал другим лицам. Поработайте с некоторыми людьми в частном порядке, прежде чем раскрыть информацию своему сообществу. Сделайте свои материалы видимыми для всех участников системы Jive. Нет  Дополнительных опций По умолчанию, Вы единственный автор этого документа.

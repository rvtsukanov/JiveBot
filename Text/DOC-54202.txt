Условия для использования интеграции 
 
 
 Как работает интеграция 
 
 Условия для использования интеграции   Для того, чтобы интеграция заработала - необходимо выполнение следующих условий:   1. Для проекта в Clarity должна быть создана папка в D2 (см. рис. 1)   Рисунок 1. - Создание папки D2 для проекта   2. В Jive должна быть создана группа для проекта и установлена связь с Clarity (то же самое проделываем для старта проектного опыта из  Jive (п. 1.2   Описание группы бизнес процессов Проектный опыт  ))  (см. рис. 2) Рисунок 2. - Связь с Clarity из  Jive   3. Использование внешнего хранилища Documentum. Для выбора необходимо выбрать настройки - настройки и в открывшемся окне выбрать в качестве внешнего хранилища D2   (см. рис. 3) .    Рисунок 3. - Выбор D2 в качестве хранилища документов   После выполнения вышеперечисленных условий в папке в D2 будет создана подпапка "Jive".   Как работает интеграция При добавлении файлов в Jive, они автоматически будут сохраняться в D2 в подпапке "Jive". Можно перемещать в другие подпапки проекта, при этом связь между системами у документа сохранится. В обычной работе с документами ничего не изменится: также можно использовать плагины Jive, выписывать документ из D2. Сохранение новой версии документа обязательно отражается в двух системах после сохранения документа.   Скоро будет реализована возможность массовой миграции, уже выложенных документов, из Jive в D2.
